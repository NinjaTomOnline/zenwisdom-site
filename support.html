Troubleshooting

Common things to check first

No meditation audio

Confirm your iPhone is not muted, media volume is up, and the track is not paused. The launch build ships with a small starter audio set, so playback issues are usually local device-state problems rather than streaming errors.

Favorites look empty

Favorites are stored locally on the device in the current build. If you reinstall the app or move to a different device, that local state does not automatically follow you.

Journal changes did not appear

Make sure the reflection body is not empty before saving. Journal history is written locally, and each saved note should appear in the day-by-day history section right away.

Premium access questions

The repo snapshot currently uses a premium scaffold and placeholder entitlement flow while live StoreKit work is pending. Support can help clarify the current release state, but full App Store billing support will arrive with the public launch.
